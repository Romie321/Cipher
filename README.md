# 🔐 Caesar Cipher
A JavaScript implementation of the classic Caesar Cipher, built for the FreeCodeCamp JavaScript Algorithms and Data Structures Certification. This project shifts each letter in a string by a fixed amount to encode or decode messages, following the rules required by the FreeCodeCamp tests.

## 🧠 What This Project Does
- Encodes and decodes text using the Caesar Cipher

- Shifts letters while preserving non‑alphabetic characters

- Handles wrap‑around from Z → A

- Passes all FreeCodeCamp test requirements for the Caesar Cipher project
